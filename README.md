# pietromarini00.github.io

Personal portfolio website hosted on GitHub Pages.

## About

This is my personal website showcasing my projects, skills, and contact information.

## Customization

To customize this website for your needs:

1. **Update `index.html`**:
   - Replace placeholder text with your own information
   - Update project descriptions
   - Add your email, LinkedIn, and other social links

2. **Modify `styles.css`**:
   - Change colors in the `:root` variables
   - Adjust layout and spacing as needed

3. **Enhance `script.js`**:
   - Add additional interactive features
   - Customize animations

## Deployment

This site is automatically deployed via GitHub Pages. Any commits to the `main` branch will be reflected on the live site.

Visit: https://pietromarini00.github.io
